# 2D-Game-Project
This project is a 2D game that features a hero character inspired by the famous bird Tweety, navigating through multiple stages filled with enemies of varying behaviors, sizes, and difficulty levels.
In stage 1 Tweety faces minor challenges as the ghost enemy only goes left and right no bullets no nothing, then in the next stages he faces wilder beasts such as the rocket releasing helicopters, the dragons and finally the gate keeper.
The game is designed to progressively challenge the player by introducing more complex enemies and mechanics in each stage.

## 🎯 Gameplay Description

* In **Stage 1**, the player faces simple enemies such as ghosts that move horizontally without attacking
* In later stages, difficulty increases with more advanced enemies, including:

  * 🚁 Helicopters that launch rockets
  * 🐉 Dragons with more aggressive behavior
  * 🚪 A final boss known as the Gatekeeper

Each level introduces new challenges, requiring better timing and strategy from the player.

# Features

* Multiple levels with increasing difficulty
* Variety of enemy types with different behaviors
* Progressive gameplay design
* Simple and engaging 2D mechanics
